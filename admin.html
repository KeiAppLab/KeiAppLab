🔑 トークンの発行手順

  1. GitHub Token設定ページへ移動 ↗
  2. [Generate new token (classic)] を選択
  3. Noteに「KEI-Admin」等と入力
  4. Expiration（期限）を設定（90 days推奨）
  5. [repo] にチェックを入れる（必須）
  6. 一番下の [Generate token] を押してコピー
⚠️ 警告：
トークンが漏洩した場合は、直ちにGitHub設定ページで該当トークンを削除（Revoke）してください。
